----------------------------------------------------------
Cómo usar el archivo requirements.txt
Guarda este texto en un archivo llamado requirements.txt en la raíz de tu proyecto (donde está README.md).

Al crear un entorno virtual nuevo o en otra máquina, ejecuta:

pip install -r requirements.txt

Esto instalará las versiones exactas de las librerías necesarias para evitar incompatibilidades.
